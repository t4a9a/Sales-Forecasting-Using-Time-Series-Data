# Sales-Forecasting-Using-Time-Series-Data
Sales Forecasting project using Random Forest on time-series data with a Streamlit web application for prediction.
# Sales Forecasting Using Random Forest

This project predicts weekly sales using a Random Forest model on time-series data.

## Features
- Data preprocessing
- Feature engineering
- Random Forest model
- Streamlit frontend

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
